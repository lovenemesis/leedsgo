YOU CAN SWITCH BETWEEN CLIENT AND THIS GUIDE BY LONG PRESSING HOME BUTTON

Press HOME button and start Email application from application launcher.

If it's your first time to set up an email account on this mobile, the "Set up email" wizard will appear right away.

For any reason the wizard does not appear, do the following steps to launch:

  1. Press MENU button and click Account settings.

  2. Press MENU button and click Add account.

In "Set up email" wizard:

  1. Fill blanks with address EMAIL and password. Click Next to continue.

  2. Select IMAP to proceed.

  3. Change IMAP server to IMAP. Make sure Port is 143 and Security type is None. Click Next to continue.

  4. Modify SMTP server to smtps.leeds.ac.uk . Make sure Port is 465 and Security type is SSL. Click Next to continue.

  5. Configure checking frequency and other options as you want. Click Next to continue.

  6. Name this account and yourself as you want. Click Done to finish.

NOW your mobile email client is configured with Leeds University email.

Thank you for using this guide.
