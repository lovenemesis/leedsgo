YOU CAN SWITCH BETWEEN CLIENT AND THIS GUIDE BY LONG PRESSING HOME BUTTON

Press HOME button and start K-9 Mail application from application launcher.

If it's your first time to set up an email account on this mobile, a wizard will appear right away.

For any reason the wizard does not appear, do the following steps in K-9 Mail to launch:

  1. Press MENU button and click More.

  2. Select Accounts button in the menu.

  3. Press MENU button again and select Add account.

In the wizard:

  1. First you will see the welcome message from K-9 team. Click Next to continue.

  2. Fill blanks with address EMAIL and password. Click Next to continue.

  3. Select IMAP to proceed.

  4. Change IMAP server to IMAP. Security type should be None whereas Authentication Type is PLAIN. Make sure Port is 143. Click Next to continue.

  5. Modify SMTP server to smtps.leeds.ac.uk . Make sure Security type is SSL(always) and Port is 465. Leave the rest to default and click Next.

  6. Configure checking frequency and number of message to display according to your need. Click Next to continue.

  7. Name this account and yourself as you want. Click Done to finish.

NOW your mobile email client is configured with Leeds University email.

Thank you for using this guide.
