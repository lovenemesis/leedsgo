YOU CAN SWITCH BETWEEN CLIENT AND THIS GUIDE BY LONG PRESSING HOME BUTTON

Press HOME button and start Mail application from application launcher.

If it's your first time to set up an email account on this mobile, you will be asked to choose an email provider.

For any reason the wizard does not appear, do the following step to launch:

  1. Press MENU button and select New Account.

In the wizard:

  1. Choose Other(POP3/IMAP) from the list.

  2. Choose Manual Setup because the auto setup does not work with university email.

  3. Select IMAP under Protocol dropdown list. Fill form with address EMAIL username USERNAME and password. Enter IMAP in IMAP server and select None under Security type. Make sure Port is 143. Click Next to continue.

  4. Change to SMTP server to smtps.leeds.ac.uk. Security type should be SSL and the Server port would be 465. Click Next to continue.

  5. Modify Account Name and Your Name as you want. Then click Finish setup.

NOW your mobile email client is configured with Leeds University email.

Thank you for using this guide.
